🔥 Overview

ReCycleX is an intelligent waste-sorting system that uses AI to:

Detect waste type
Auto-categorize
Upload it to a marketplace
Allow users/recyclers to buy & sell
Track reward points for each recycle

📦 Setup & Installation
1️⃣ Clone Repo
git clone https://github.com/Jannveee/recyclex.git
cd recyclex
🖥️ Client Setup
cd client
npm install
npm run dev
🖧 Server Setup
cd ../server
npm install
npm start
🌐 Environment Variables

Create .env files in:

Client → client/.env
VITE_API_URL=your_backend_url_here
Server → server/.env
MONGO_URI=your_mongodb_connection_string
